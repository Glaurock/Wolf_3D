# Wolf_3D


## Compiling

Running make will compile all the libraries and the project. 

## Usage
./wolf3d <map>

COMMANDS :
W, A, S, D to move
Q, E to straf
+, - to change FOV
P to pause the game
SPACE to fire
ENTER to reset
ESC to exit
